Architectures

Architectures are the structure of the given neural network. These architectures are then given input and used for training.

Models

After creating an architecture and training it on a node, the resulting models are stored here. These models are used for classifying images.

Data


Images can be uploaded (in zip files) here to be used for training our models.

Labeled Data


Images from "Data" can be composed into classes here. These classes are then used for training models for image classification.
